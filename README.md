# card-shuffle-ui

Простой проект: 100 карточек с заданиями. При нажатии "Перемешать" идёт плавная видимая анимация перемешиваний, затем реальная перестановка и выпадение одной карты — задача показывается в модалке. При подтверждении карта удаляется (не выпадёт повторно).  

## Запуск
1. Помести файлы `index.html`, `style.css`, `tasks.js`, `script.js` в корень репозитория.
2. Убедись, что `tasks.js` содержит массив `tasks` из 100 элементов.
3. Размести на GitHub и включи GitHub Pages (branch `main`, root folder).

## Что в коде
- `style.css` — стили и ключевые keyframes для видимой фазы перемешивания.
- `script.js` — логика видимого shuffle (CSS) + реального shuffle (DOM + FLIP) + модалка и удаление карты.
